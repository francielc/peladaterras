CONFIGURAÇÃO DO BANCO DE DADOS
PASSO 1 — CRIAR PROJETO FIREBASE (GRATUITO)

Acesse console.firebase.google.com → "Criar projeto" → dê um nome (ex: pelada-fc) → finalize. Pode desativar o Google Analytics.

PASSO 2 — ATIVAR O BANCO DE DADOS (FIRESTORE)

No menu lateral: Firestore Database → Criar banco de dados → Modo de teste → escolha a região southamerica-east1 (São Paulo) → Concluir.

PASSO 3 — PEGAR CONFIGURAÇÃO DO PROJETO

Menu lateral → ⚙️ Configurações do projeto → Seus aplicativos → Adicionar app → Web (</>) → registre com qualquer nome → copie o objeto firebaseConfig que aparecer.

PASSO 4 — COLAR ABAIXO

Cole o objeto { "apiKey": "...", ... } no campo abaixo. Isso é salvo no navegador — você só faz isso uma vez por dispositivo.
